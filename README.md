# ChT-MINI
A high-performance, cinematic "Business Card" landing page inspired by the Haikyuu!! aesthetic. This project blends elite UI/UX principles like Glassmorphism and Bento Grid layouts with industry-standard animations.



Key Features
Cinematic Animations: Powered by GSAP (GreenSock) for smooth entrance staggers and floating hero elements.

Interactive Parallax: The background "KARASUNO" text and character assets respond to mouse movement for a 3D depth effect.

Modern Bento Grid: A "Most Popular" section featuring a responsive grid layout with hover-triggered expansion.

Glassmorphism UI: Custom-built glass cards using backdrop-filter for a premium, futuristic feel.

Fully Responsive: Optimized for everything from high-res monitors to mobile devices using Tailwind CSS.

Functional Modals: Built-in dynamic modal system for project details and contact triggers.

🛠️ Tech Stack
Frontend: HTML5, Tailwind CSS (Utility-first styling)

Animation Engine: GSAP 3.12.2 (GreenSock)

Typography: Google Fonts (Oswald for headers, Poppins for body)

Icons/Avatars: DiceBear API

🎨 Design Philosophy
The goal was to move away from static, boring business sites. By using a Diagonal Split layout and High-Contrast Orange (#FF6B00) accents, the site mimics the energy of a volleyball match. Every element—from the custom scrollbar to the micro-interactions on the buttons—is designed to keep the user engaged.
